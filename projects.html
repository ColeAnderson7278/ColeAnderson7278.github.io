This is a simple side scrolling game where you dodge falling rain drops. I made it using basic JavaScript, HTML, and CSS. It may look simple, but it showed me a lot because I learned about making background data that virtually is the game. Then, after this data was established, use the data to play the game using other functions that manipulates the data accordingly. After this I had to us my HTML and CSS to show this data in a way the user could understand and enjoy. These to aspects taught me a lot about how the back of my program has to communicate with the front, and then provide something interesting and useful to the user.
